#  Number Guessing Game in Java 
A simple console-based Number Guessing Game developed in Java to practice basic programming concepts and data types. The player has 3 lives to guess a secret number between 1 and 10. If the correct number is guessed, the player wins and earns points; otherwise, the game ends when all lives are exhausted. Features User input using Scanner Data types: byte, short, int, and boolean Conditional statements (if-else) Looping with while Score tracking system Simple and beginner-friendly code Technologies Used Java VS Code / Eclipse / IntelliJ IDEA JDK 8 or above
